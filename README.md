# Employe_api# Employee Management API (Flask + SQLAlchemy)

A simple RESTful API built with Flask and SQLAlchemy to manage employees.

## Features
- Add new employee
- Get all employees
- Get employee by ID
- Update employee details
- Delete employee

## Tech Stack
- Python, Flask
- SQLite, SQLAlchemy
- Postman for testing

## Run Locally
```bash
pip install -r requirements.txt
python app.py
